# Error DB — 錯誤模式資料庫

> 追蹤「錯誤模式」（Pattern），不是具體原句；原句案例放 `10_knowledge_base/error_corrections/`。
> 一個 Pattern 可對應多個原句案例。
> 建立條目的門檻：同一類錯誤第 2 次被觀察到（見 CLAUDE.md 原則 2）。
>
> - Category：`Grammar` `Pronunciation` `Gender` `Article` `Conjugation` `Spelling` `Listening` `Expression`
> - Confidence：Low（Count = 2）/ High（Count ≥ 3，同時加入 dashboard 的 Active Weaknesses）
> - Status：Active / Improving / Resolved
> - Evidence：連到 evidence_log.md 的日期與 error_corrections/ 的案例檔
