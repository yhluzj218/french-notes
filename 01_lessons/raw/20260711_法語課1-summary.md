## 法語字母與發音教學

### 一、法語字母介紹

- 法語字母共有 **26 個字母**（l'alphabet）
- 老師帶領全班跟讀每個字母，學生跟著重複朗讀

### 二、字母認讀練習

- 老師隨機點字母，請學生說出讀音
- 點名 Janet、Wendy、Sandra、Kiki 等學生個別作答
- 練習字母：N、O、Q、U、E、X、W、V、Y、R、L、A、C、P、T、Z、K、G、F

### 三、課本活動：拼音辨識（Page 13，Activité 1）

- 老師播放影片，學生**聽字母拼音、辨識法語人名**
- 活動內容：聽到拼音後，說出或寫出完整的名字

| 題目 | 拼音 | 答案 |
|------|------|------|
| 例題 | L-I-N-A | Lina |
| A | A-Y-A | Aya |
| B | B-I-C-T-O-R / V-I-C-T-O-R | Victor |
| C | （含 E、M、C 等字母） | Emma（推測） |
| D | J-U-L-I-A | Julia |
| E | G-A-B-R-I-E-L | Gabriel |
| F | F-O-S-C-A-R / O-S-C-A-R | Oscar |

- 過程中老師逐步引導學生修正錯誤，請不同學生補充遺漏的字母

### 四、自我介紹時拼出姓名

- 老師示範句型：**「Mon prénom c'est Anne. Ça s'écrit A-N-N-E.」**
- 輪流請每位學生拼出自己的名字：
  - Wendy：W-E-N-D-Y
  - Janet：J-A-N-E-T
  - Elsa：E-L-S-A
  - Sandra：S-A-N-D-R-A

### 五、法語母音發音練習

老師帶讀，學生跟讀，逐一練習以下母音音素：

| 音素 | 拼寫方式 | 例字 |
|------|----------|------|
| **[i]** | i, î, naïf | dîner, naïf, lisser |
| **[e]** | é, er, ez | aller, blé |
| **[ɛ]** | è, ê, ë, ai | très, forêt, Noël, belge |
| **[a]** | a, â | avec, ami, Anne |
| **[o]** | au, ô, eau | Maud, diplôme, chaud |
| **[u]** | ou | vous, bichou |
| **[y]** | u | tu, du, rue, une |
| **[ø]** | eu, œu | je, deux |
| **[ɛ̃]** | in, ain, un | lundi, simple, parfum |
| **[ã]** | an, en, am, em | bande, enfant |
| **[ɔ̃]** | on, om | ombre, longue |

- 老師點名學生個別發音，糾正發音位置（如 **[ø]** 需要「舌頭往後」）
- 課堂結束前宣布休息 10 分鐘
