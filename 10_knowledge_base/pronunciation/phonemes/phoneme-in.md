Phoneme: "/ɛ̃/"
Type: 鼻母音
Description: 鼻母音——先做 [ɛ]（嘴垂直打開）再讓氣同時從鼻子出去；嘴巴不收尾、舌頭不碰上顎
Spellings: "in, im, ain, aim, ein, (i)en, un, um"
Examples:
  - "bien /bjɛ̃/"
  - "pain /pɛ̃/"
  - "vin /vɛ̃/"
Confusions: "/ɑ̃/（enfant）、/ɔ̃/（ombre）——鼻母音三分；/ɛ/（非鼻化：plaine vs plein）"
Source: Lesson 01（2026-07-11）；500 exercices p.77（[ɛ]–[ɛ̃]）、p.94（三鼻母音綜合）

## 代表字

| 單字 | IPA | 備註 |
|---|---|---|
| bien | /bjɛ̃/ | Très bien（每堂課都在說） |
| pain | /pɛ̃/ | |
| vin | /vɛ̃/ | 跟數字 vingt /vɛ̃/ 同音 |
| lundi | /lœ̃di ~ lɛ̃di/ | un 拼法；現代法語多數人已併入 /ɛ̃/ |
| demain | /dəmɛ̃/ | À demain（道別語） |

## 常見拼法

| 拼法 | 例字 | 備註 |
|---|---|---|
| in | vin, lapin | 最常見 |
| im | simple, impossible | 後接 b/p 時用 im |
| ain | pain, demain | |
| aim | faim | 後接 b/p |
| ein | plein, peinture | |
| (i)en | bien, chien | i + en 時 en 念 /ɛ̃/ 不是 /ɑ̃/ |
| un / um | lundi, parfum | 傳統是 /œ̃/，現代法語多併入 /ɛ̃/，A1 先當同一個音 |

⚠️ 陷阱：這些拼法後面**再接母音或再一個 n/m 時就不鼻化**——plein /plɛ̃/ 但 pleine /plɛn/、vin /vɛ̃/ 但 vinaigre 的 vi-n 分開念。

## 易混音（minimal pairs）

- /ɛ̃/ vs /ɑ̃/ vs /ɔ̃/：demain / enfant / ombre（鼻母音三分，500 ex p.94）
- /ɛ̃/ vs /ɛ/：plein / plaine、vin / vaine（鼻化 vs 非鼻化，500 ex p.77）
