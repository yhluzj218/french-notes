---
Phoneme: "/j/"
Type: 半母音
Description: 半母音 yod——像很短的「一」滑進旁邊的母音；= 英文 yes 的 y
Spellings: "i+母音, il/ill, y+母音"
Examples:
  - "bien /bjɛ̃/"
  - "fille /fij/"
  - "yoga /jɔga/"
Confusions: "/ʒ/（yod vs j 字母音，500 ex p.154）"
Source: 500 exercices p.154
Learning Status: New
Anki Status: Pending

## 常見拼法

| 拼法 | 例字 | 備註 |
|---|---|---|
| i + 母音 | bien, panier, hier | i 變成滑音 |
| -ill- | fille, famille, cuillère | |
| -il 字尾 | travail, soleil | |
| y + 母音 | yoga, yeux | |

⚠️ **ville /vil/、mille /mil/ 是例外**——ill 念 /il/ 不是 /ij/

## 易混音（minimal pairs）

- /j/ vs /ʒ/：yeux 類 vs jeu 類（500 ex p.154），別把 fille 念成 fi-ge
