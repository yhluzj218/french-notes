## 常見拼法

| 拼法 | 例字 | 備註 |
|---|---|---|
| é | café, décidé, école | 最常見 |
| -er 動詞字尾 | écouter, parler | r 不發音 |
| -ez | chez, vous allez | |
| -es（單音節） | les, des, mes, ses | |
| -et 字尾 | et（和） | |

## 易混音（minimal pairs）

- /e/ vs /i/：des / dis、ses / si（**evidence log 7/13 弱點，本週日 p.45 主攻**）
- /e/ vs /ɛ/：décidé / décider 尾音、été / était（差別 = 嘴巴開多大）

## 糾正記錄

- 自察（2026-07-13）：中英文都無純 /e/，最難的高前母音；[ɛ] 可借英文 bed 但 /e/ 沒得借

## 自創記憶掛鉤（Connie 版）

- 像中文「誒」但嘴更閉、嘴角橫拉（微笑狀），**定住不滑動**（不要滑成英文的 ay）
- 字母 G 的名字 = /ʒe/，尾音就是這個 /e/
