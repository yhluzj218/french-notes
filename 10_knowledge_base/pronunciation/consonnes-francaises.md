Rule: 子音拼字 → 發音規則（c / ç / g / j / s / ss / ch / gn 等）
IPA: "[k] [s] [g] [ʒ] [z] [ʃ] [ɲ] [b] [p] [d] [t] [f] [v] [ʁ]"
Description: 同一個字母依後面接的字母決定發音；本條收 Lesson 02 教過的子音規則與例字
Examples:
  - "ce / ci / cy → [s]：Céline, ceci"
  - "ca / co / cu → [k]：café, cou"
  - "ge / gi → [ʒ]：page, gîte, pigeon"
  - "ga / go → [g]：gazon, gare"
  - "poisson [pwasɔ̃] vs poison [pwazɔ̃]"
Exceptions: "ç 永遠 [s]（不受後面字母影響）：leçon, garçon"
Source: Lesson 02（2026-07-18），課本 p.14–15 範圍
Learning Status: New
Anki Status: Pending

## 規則總表

| 拼字 | 發音 | 條件 | 例字（課堂） |
|---|---|---|---|
| c + e/i/y | [s] | 後接前母音 | Céline, ceci, cy… |
| c + a/o/u | [k] | 其他 | café, cou（qu 也是 [k]） |
| ç | [s] | 永遠 | leçon, garçon |
| g + e/i | [ʒ] | 後接前母音 | page, gîte, pigeon |
| g + a/o/u | [g] | 其他 | gazon, gare |
| j | [ʒ] | 永遠 | jupe, bonjour |
| s（母音之間） | [z] | 單個 s 夾在兩母音間 | poison, oiseau, rose |
| ss | [s] | 永遠 | poisson |
| ch | [ʃ] | 一般情況 | chat, chapeau |
| gn | [ɲ] | | agneau, champagne |

## 塞音/擦音對（中文母語者注意：法語有清濁對立）

| 對比 | 例字 |
|---|---|
| b [b] vs p [p] | bébé / pépé、hibou / poule |
| d [d] vs t [t] | dents / temps、midi、malade |
| f [f] vs v [v] | fée, œuf, éléphant / vélo |
| s [s] vs z [z] | poisson / poison、zéro |

- 老師警句：« Je mange du **poisson**, et pas du **poison**. »（吃魚不是吃毒藥）
- r [ʁ]（rose）：喉音，全班都覺得難；Phase 1 後段照 500 exercices p.162 專攻
