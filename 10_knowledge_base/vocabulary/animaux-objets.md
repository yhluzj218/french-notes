這組單字的價值不在單字本身（多為 ★ 認識即可），在**每個字示範的規則**：

| 法語 | IPA | 中文 | 掛的規則 |
|---|---|---|---|
| le chat | /ʃa/ | 貓 | ch = /ʃ/；字尾 t 不發音 |
| le poisson | /pwa.sɔ̃/ | 魚 | **ss = /s/**；對比 poison /pwa.zɔ̃/ 毒藥 |
| la poule | /pul/ | 母雞 | p/b 對比例字（poule vs boule） |
| l'abeille | /a.bɛj/ | 蜜蜂 | **eille = /ɛj/**（Marseille 同款） |
| la bague | /baɡ/ | 戒指 | **gu 的 u 是護衛字母**（保住 /g/） |
| neuf | /nœf/ | 9／新的 | 字尾 f 發音；neuf heures /nœ.vœʁ/ f→/v/ 特殊連音 |
| l'école | /e.kɔl/ | 學校 | é=/e/；o+子音=/ɔ/ |
| attention | /a.tɑ̃.sjɔ̃/ | 注意 | **ti = /s/**（-tion 規則） |
| la musique | /my.zik/ | 音樂 | **s 夾母音間 = /z/** |
| le vélo | /ve.lo/ | 腳踏車 | 全規則字；口語不說 bicyclette |
| l'avion | /a.vjɔ̃/ | 飛機 | ion = /jɔ̃/；A comme Avion |
| le wagon | /va.ɡɔ̃/ | 車廂 | **w = /v/**（德源；對比 wifi 的 w=/w/） |
| l'agneau | /a.ɲo/ | 小羊 | gn = /ɲ/；eau = /o/ |
| le champagne | /ʃɑ̃.paɲ/ | 香檳 | ch=/ʃ/ + am=/ɑ̃/ + gne=/ɲ/ 三規則一字 |
| le panier | /pa.nje/ | 籃子 | **nier = /nje/ 不是 /ɲ/**；-ier = /je/ |
| le pigeon | /pi.ʒɔ̃/ | 鴿子 | **geo 的 e 是護衛字母**（保住 /ʒ/） |
| la page | /paʒ/ | 頁 | ge = /ʒ/ |
| le gîte | /ʒit/ | 民宿 | gi = /ʒ/；î 不影響發音 |
| le rhinocéros | /ʁi.nɔ.se.ʁɔs/ | 犀牛 | rh = /ʁ/；⚠ **字尾 s 要發音（例外字）** |
