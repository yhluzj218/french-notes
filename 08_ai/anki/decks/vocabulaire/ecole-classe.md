# Deck：學校與課堂（5 卡）

> 位置即狀態：這裡 = 還沒錄入 Anki。

### l'école — /e.kɔl/
學校｜陰性｜★★★
- 讀音：é = /e/；o 接子音 = /ɔ/（開口）
- 連音：l'école（le 縮寫）、les écoles /le.ze.kɔl/

### la page — /paʒ/
頁｜陰性｜★★★
- 讀音：ge = /ʒ/；課堂高頻字（page 4, page 14…）

### attention — /a.tɑ̃.sjɔ̃/
注意｜陰性・感嘆詞｜★★★
- 讀音：en = /ɑ̃/；**ti = /s/**（-tion 規則）；tt 發一個 /t/
- 用法：Attention !（小心！）——老師課堂高頻警語

### le nombre — /nɔ̃bʁ/
數字、數目｜陽性｜★★
- 讀音：om = /ɔ̃/（接 b/p 用 om 拼）；字尾 -bre 的 e 輕到幾乎沒有

### la voyelle — /vwa.jɛl/
母音｜陰性｜★★
- 讀音：v 牙齒碰下唇；oy + 母音 = /waj/；-elle = /ɛl/
- 用法：les voyelles = A, E, I, O, U, Y（判斷 liaison 用）
