# Deck：數字（10 卡）

> 位置即狀態：這裡 = 還沒錄入 Anki。完整 0–31 對照表見 KB 的 nombres-0-31。

### zéro — /ze.ʁo/
0｜★★★
- 讀音：z 聲帶要震（不是 /s/）

### un — /œ̃/
1｜★★★
- 讀音：傳統 /œ̃/，現代多數人念 /ɛ̃/，都對
- 連音：un + 母音必連 → un euro /œ̃.**n**ø.ʁo/、un ami /œ̃.**n**a.mi/

### deux — /dø/
2｜★★★
- 讀音：eu = /ø/；x 不發音
- 連音：deux + 母音 → x 發 /z/：deux euros /dø.**z**ø.ʁo/

### trois — /tʁwa/
3｜★★★
- 讀音：oi = /wa/；s 不發音
- 連音：trois euros /tʁwa.**z**ø.ʁo/

### six — /sis/
6｜★★★
- 讀音三態：單獨 /sis/；+ 子音 /si/（six dollars）；+ 母音 /siz/（six euros）

### sept — /sɛt/
7｜★★★
- 讀音：p 不發音；**t 要發音**（少數字尾 t 發音的字）

### neuf — /nœf/
9／新的｜數字・形容詞｜★★★
- 讀音：eu + f = /œ/（開口）；字尾 f 要發音
- ⚠ 特殊連音：neuf heures（9點）、neuf ans（9歲）的 f 變 /v/ → /nœ.vœʁ/ /nœ.vɑ̃/
- 一字兩義：9；「新的」（une voiture neuve）

### dix-huit — /di.zɥit/
18｜★★
- 讀音：dix 的 x 在此連成 /z/；ui = /ɥi/；字尾 t 發音

### vingt — /vɛ̃/
20｜★★★
- 讀音：gt 都不發音
- 變化：vingt-deux /vɛ̃t.dø/ 的 t 復活；vingt euros /vɛ̃.**t**ø.ʁo/ 連音

### trente — /tʁɑ̃t/
30｜★★
- 讀音：en = /ɑ̃/
- 變化：trente-et-un /tʁɑ̃.te.œ̃/（21、31 才有 et）
